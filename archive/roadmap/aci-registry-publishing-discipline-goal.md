# ACI Registry Publishing Discipline Goal

Status: Complete

## Goal

Define the bounded registry-publishing discipline that `ACI` must satisfy before any package registry publication is allowed.

## Complete When

- one bounded registry-publishing contract exists
- release-prep reading order explains where registry publication starts and stops
- publish decision gates, evidence, and hold lines are explicit
- reader guidance explains what is still intentionally out of scope

## Out of Scope

- actual package upload
- credential setup on any live registry
- public release announcement
- downstream rollout

## Failure Conditions

- registry publication is implied without an explicit gate
- private proof and public registry steps are blurred together
- release discipline still depends on oral explanation
