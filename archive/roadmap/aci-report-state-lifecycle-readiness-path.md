# ACI Report State Lifecycle Readiness Path

Status: Complete

## Path

1. inventory the current state language
2. define the minimum generic state lifecycle
3. update finding and report contracts
4. update examples and reader guidance
5. verify repeated-state readability

## Stop Conditions

- lifecycle states start requiring downstream persistence
- lifecycle overlaps too heavily with baseline/suppression/waiver concepts
- a state requires organization-specific policy
