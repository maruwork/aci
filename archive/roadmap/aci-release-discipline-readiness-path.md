# ACI Release Discipline Readiness Path

Status: Complete

## Path

1. inventory current release-discipline language
2. define the bounded release discipline model
3. implement the common release discipline packet
4. document release-prep boundaries and reading route
5. verify the reviewed release surface

## Stop When

- release guidance starts depending on one host provider
- versioning policy requires decisions that belong to owner-specific publication timing
- the common shelf starts claiming live release operations
