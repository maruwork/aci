# ACI Release Notes Discipline Goal

Status: Complete

## Goal

Define the bounded release-notes discipline that `ACI` must satisfy before any release notes are considered ready.

## Complete When

- one bounded release-notes discipline contract exists
- release-prep reading order points to that contract
- required release-note sections, evidence, and hold lines are explicit
- reader guidance explains what remains outside this shelf

## Out of Scope

- hosted release-page formatting
- public announcement copy
- marketing messaging
- downstream rollout

## Failure Conditions

- release-note content is still implied instead of written
- release-prep docs disagree about what a reviewed release note must contain
- release-note readiness still depends on oral explanation
