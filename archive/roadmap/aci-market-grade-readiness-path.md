# ACI Market-Grade Readiness Path

Status: Complete

## Path

1. inventory the current product surface from install to long-term operation
2. compare that surface against market-grade expectations
3. split the gaps into bounded product lanes
4. turn each lane into concrete next-wave work
5. review whether any hidden prerequisite is still missing

## Product Lanes

- installation and setup
- runtime and execution model
- triage and state management
- baseline, suppression, and waiver handling
- CI and automation contract
- trust and regression safety

## Stop Conditions

- a gap requires external business decisions rather than tool-body design
- a gap belongs to a downstream project integration rather than `ACI`
- a proposed lane is too broad to become executable work
