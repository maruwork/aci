# ACI Reliability Fixture Suite Readiness Path

Status: Complete

## Path

1. define the minimum fixture scope
2. add the fixture shelf and expected data
3. implement the fixture check
4. document the fixture contract
5. verify repeatable fixture execution

## Stop Conditions

- fixture scope requires downstream runtime state
- a fixture becomes too broad to stay stable in the common shelf
- a required behavior belongs to the report state lifecycle wave
