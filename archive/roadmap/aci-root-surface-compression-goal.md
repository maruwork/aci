# ACI Root Surface Compression Goal

Status: Complete

## Goal

`ACI` の root を、利用者が最初に読む面としてさらに薄くし、maintainer history と release-prep 文書を直接浴びなくて済む状態へ持っていく。

## Completion State

- root README から見える主導線が短い
- release-prep 文書群は専用 index から辿る形になる
- maintainer history 文書群は専用 index から辿る形になる
- root 直下の大量文書が利用者向け導線を邪魔しない

## Out Of Scope

- report contract の schema 変更
- domain pack 実装変更
- 公開操作
