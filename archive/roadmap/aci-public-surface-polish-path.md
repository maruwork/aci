# ACI Public Surface Polish Path

Status: Complete

## Path

1. public-facing surface を棚卸しする
2. 利用者向け入口と内部履歴入口を分離する
3. sample / smoke / report 説明の温度差を潰す
4. final review で public-facing consistency を確認する

## Stop Points

- public-facing file を削ることで既存 reading route が壊れる時
- internal document を public surface から外すだけでは済まず authority 変更が必要な時

## Dependency Rule

- 入口修正の前に public-facing file inventory が必要
- sample 修正の前に target tone が必要
- final review の前に README / smoke / sample の追従が必要
