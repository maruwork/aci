# ACI Domain Pack Boundary Goal

Status: Complete

## Goal

`ACI` の common 正本と optional domain pack の境界を、将来 domain pack が増えても崩れない状態へ持っていく。

## Completion State

- common 正本に残すものと domain pack に置くものの基準が明確
- `domains/` の入口が読みやすい
- `Pier` pack の bridge docs と common docs の責務が明確
- 新しい domain pack を追加する時の最小 route が分かる

## Out Of Scope

- 新しい domain pack 実装
- downstream project への runtime 接続
- 公開操作
