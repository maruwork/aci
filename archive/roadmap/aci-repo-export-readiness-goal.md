# ACI Repo Export Readiness Goal

Status: Complete

## Goal

Make `epo root` ready to be lifted into a standalone public repository without losing meaning, entry clarity, or boundary rules.

## Complete When

- standalone-repo expectations are explicit
- the current shelf is audited as a standalone repo
- missing repository-facing assets are added or explicitly named
- blocker status is recorded

## Closure

- standalone-repo expectations are explicit in `docs/release/REPO_EXPORT_GUIDE.md` and `docs/release/REPO_EXPORT_CHECKLIST.md`
- repository-facing public paths are repo-relative
- blocker status is `none` inside shelf scope
