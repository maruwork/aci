# ACI Registry Upload Runbook Goal

Status: Complete

## Goal

Define the bounded runbook that a maintainer must follow immediately before any live `ACI` package registry upload is attempted.

## Complete When

- one bounded registry upload runbook exists
- release-prep reading order points to that runbook
- upload sequence, evidence, and stop lines are explicit
- reader guidance explains what remains outside this shelf

## Out of Scope

- actual registry upload
- credential setup
- provider-specific release page actions
- downstream rollout

## Failure Conditions

- upload preparation is still implied instead of written
- release-prep docs disagree about the order of upload steps
- upload readiness still depends on oral explanation
