# ACI Baseline Suppression And Waiver Readiness Path

Status: Complete

## Path

1. inventory current baseline/waiver language in `ACI`
2. define the minimum generic model
3. decide the common-shelf artifact shape
4. update report and finding contracts where needed
5. add bounded examples and reading guidance

## Stop Conditions

- the model starts requiring downstream storage decisions
- exception handling drifts into organization policy rather than generic tool design
- one concept cannot be kept separate from the others
