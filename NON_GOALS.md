# ACI Non-Goals

Status: Active

## Purpose

Reduce over-expectation for first-time readers.

## Non-Goals

- `ACI` is not a full project runtime gate by itself
- `ACI` is not a replacement for human approval
- `ACI` is not a packaged downstream CLI distribution in this shelf
- `ACI` does not own project-local trigger wiring
- `ACI` does not own project-local DB writeback
- `ACI` does not replace front-stage security or commit policy tooling

## What This Shelf Does Own

- generic inspection catalog
- normalized finding helpers
- optional domain-pack selection
- reporting contracts
- common examples and public samples
