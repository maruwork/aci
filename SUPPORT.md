# ACI Support

Status: Active

## Before Opening An Issue

1. Read `README.md`.
2. Run `python python/aci_public_smoke.py`.
3. Confirm whether the problem belongs in generic ACI, an optional domain pack, or downstream integration.
4. Check whether the expected output is already described in `report/` contracts or `core/` specs.

## Where To Ask

- reproducible defects: open a bug report
- new capability proposals: open a feature request
- private repository preparation questions: include the exact step from `PRIVATE_HOST_BOOTSTRAP_RUNBOOK.md`

## What To Include

- command used
- relevant file path
- expected behavior
- actual behavior
- smallest useful log or artifact excerpt
