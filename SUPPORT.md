# ACI Support

Status: Active

## Before Opening An Issue

1. Read `README.md`.
2. Run `python shared/python/aci_public_smoke.py`.
3. Confirm whether the problem belongs in generic ACI, an optional domain pack, or downstream integration.
4. Check whether the expected output is already described in `shared/report/` contracts or `shared/core/` specs.
5. Confirm that the version and install mode are within generic support scope.

## Where To Ask

- reproducible defects: open a bug report
- new capability proposals: open a feature request
- support scope questions: include the exact lifecycle window and install mode

## What To Include

- command used
- version or release tag
- install mode
- relevant file path
- expected behavior
- actual behavior
- smallest useful log or artifact excerpt
